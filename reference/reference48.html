48. Generators

generatorPrompt

  • The prompt used for delimiting generator.

Generator

  • (type Class)
    (extends Obj)
    (attributes value continuation)
     
    Instances of this class are yielded.

\*

  • (\* parametersTree . forms)
    (type fexpr)
     
    Returns an anonymous generator function with the given parametersTree, when executed, it will return a generator object with forms as body.
    In the body of the generator object can be called
    - the (yield* . value) function to pause the generator and returning an optional value,
    - the (yield** generator) function to delegate the iteration to another generator.

\**

  • (\** . forms)
    (type fexpr)
     
    Returns an generator object with forms as body.
    In the body of the generator object can be called
    - the (yield* . value) function to pause the generator and returning an optional value,
    - the (yield** generator) function to delegate the iteration to another generator.

next*

  • (next* generator)
    (next* generator value)
    (type fexpr)
     
    Resume a generator yielded, the optional value or #inert will be the value returned from the yield* resumed.
