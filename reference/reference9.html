9. Obj & Box

new

  • Return a new instance of class with the given value or bindings.
     
    (new boxClass)
    (new boxClass value)
    (new objClass . bindings)
    (new objClass string . bindings)
    (new objClass string throwableObject . bindings)

defNew

  • Defines into the current environment a named instance name of the given class and value or bindings.
     
    (defNew name boxClass)
    (defNew name boxClass value)
    (defNew name objClass . bindings)
    (defNew name objClass string . bindings)
    (defNew name objClass string throwableObject . bindings)
    (type macro)
