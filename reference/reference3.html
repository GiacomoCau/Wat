3. Definition Forms

  • The forms defMacro defVau def\ defde\ def*\ rec\ let1\ let1rec\ let\ and letrec\ have two equivalent syntax
     
            (_ name parameterTree . forms)
            (_ (name . parameterTree) . forms)
     
    The forms rec rec\ let1rec let1rec\ letrec and letrec\ pre-initialize all bindings to #inert before evaluating and binding the values or functions.

defMacro

  • (defMacro name parameterTree . forms)
    (defMacro (name parameterTree) . forms)
    (defMacro name bindResult parameterTree . forms)
    (defMacro (name parameterTree) bindResult . forms)
    (type macro)
     
    Defines into the current environment the named macro name with the given parameterTree, bindResult and forms as body.

defVau

  • (defVau name parameterTree environmentParameter . forms)
    (defVau (name parameterTree) environmentParameter . forms)
    (defVau name bindResult parameterTree environmentParameter . forms)
    (defVau (name parameterTree) bindResult environmentParameter . forms)
    (type macro)
     
    (syntax bindResult (or #ignore #inert :rhs :prv :cnt))
     
    Defines into the current environment the named fexpr name with the given parameterTree, bindResult, environmentParameter and forms as body.

defConstant

  • (aliasof def)
     
    Defines into the current environment the named constant name with the given value.
    This is mostly for documentation purposes, as constants are still mutable.

def*

  • (def* definiendTree . values)
    (def* definiendTree bindResult . values)
    (type macro)
     
    (syntax bindResult (or #ignore #inert :rhs :prv :cnt))
     
    Defines into the current environment the resulting bindings from the match of the definiendTree against values, signals an error otherwise.

set!*

  • (set!* definiendTree . values)
    (set!* definiendTree bindResult . values)
    (type macro)
     
    (syntax bindResult (or #ignore #inert :rhs :prv :cnt))
     
    Update into the environment the resulting bindings from the match of the definiendTree against values, signals an error if a binding is not defined.

def\

  • (def\ name parameterTree . forms)
    (def\ (name parameterTree) . forms)
    (def\ name bindResult parameterTree . forms)
    (def\ (name parameterTree) bindResult . forms)
    (type macro)
     
    (syntax bindResult (or #ignore #inert :rhs :prv :cnt))
     
    Defines into the current environment the named function name with the given parameterTree, bindResult and forms as body.
