2. Macro

expandMacro

  • A integer to discriminate when to evaluate (== 0) or expand (> 0) for n level a macro.
    Used from expand and makeMacro.

makeMacro

  • (makeMacro expander)
    (type function)
     
    Return a macro from an expander operator.
    A macro is an operator that receives an operand and produces a form
    (by calling the expander with the operand as argument)
    that is then evaluated in place of the operand.

macro

  • (macro parameterTree . forms)
    (type macro)
     
    Return an anonymous macro with the given parameterTree and forms as body.

expand

  • (expand form)
    (expand nLevel form)
    (type macro)
     
    Expands a macro call for 1 or n level rather than evaluates it.
