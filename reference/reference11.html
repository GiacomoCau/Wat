11. List

  • (type Class)
    (extends Cons)
     
    The last `cdr' of lists is alwais #null.

append

  • (append list . rest)
    (type function)
     
    (syntax rest (or (list/object) (list . rest)))
    (derivation (\ (list list/object . rest) (apply** (rec\ (append list . rest) (if (null? rest) list (%append list (apply append rest)))) list list/object rest)))
     
    Return the append of list/object to the lists.
    The firsts one lists must be all proper list and are copied.
    The last one list/object is not copied (and doesn't even have to be a list).
    It becomes the `cdr' of the final cons of the copied lists,
    or is returned directly if list is empty.

len

  • (len list)
    (type function)
     
    Return the number of elements in the list.

list?

  • (list? object)
    (type function)
     
    (derivation (type? object List))
     
    Return #true if the object is a list, #false otherwise.
