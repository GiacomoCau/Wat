40. Generic Functions

defGeneric

  • (defGeneric name receiverName . parameters)
    (defGeneric (name receiverName) . parameters)
    (type fexpr)
     
    Define a new generic function with the given name.
    receiverName, parameters, and properties are currently ignored.

defMethod

  • (defMethod ((name (receiverName class) . parameters) . forms))
    (defMethod (name ((receiverName class) . parameters) . forms))
    (type fexpr)
     
    Define a new method for the generic function name specialized for class.
