7. Env & Obj

bound?

  • (bound? attribute object)
    (type function)
     
    (syntax attribute (or Symbol Keyword String))
    (syntax object (or Env Obj))
     
    (derivation (@isBound object attribute))
     
    Return #true if the attribute is bound in the environment or obj, #false otherwise.
