49. Auto Increment/Decrement and Assignement Operators

++

  • (++ number)
    (++ box)
    (++ obj field)
    (type fexpr)
     
    Return the incremented value of Number, Box and Obj.

--

  • (-- number)
    (-- box)
    (-- obj field)
    (type fexpr)
     
    Return the decrenented value of Number, Box and Obj.

assignOp

  • (assignOp operator)
    (type function)
     
    Return a fexpr which returns the value assigned to Box, ObjEnv and Object after applying the operator.

$=

  • (def $= (assignOp %$))

+=

  • (def += (assignOp %+))

-=

  • (def -= (assignOp %-))
